# Falling Cat Fire

## Description
Falling Cat Fire is a fast-paced and easy-to-play casual game. Face different enemies, collect power-ups, trigger special events, and unlock achievements while putting your reflexes to the test.

## Features
- 12 different enemies
- 3 unique power-ups
- 2 special in-game events
- 8 achievements to unlock

## Engine
- Godot Engine
